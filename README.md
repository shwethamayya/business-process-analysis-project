# Business Process Analysis & Improvement Project

## Project Overview
This project analyzes an existing customer support process, identifies inefficiencies, and proposes an improved TO-BE process using basic business analysis techniques.

## Process Chosen
Customer Support Ticket Handling Process

## What I Did
- Studied the existing AS-IS process
- Identified bottlenecks and pain points
- Designed an improved TO-BE process
- Compared KPIs before and after improvement

## Files Included
- **Excel**: AS-IS vs TO-BE data (150+ tickets, KPIs, metrics)
- **PowerPoint**: Process flow, bottlenecks, improvements, and business impact

## Tools Used
- Microsoft Excel
- Microsoft PowerPoint
- Business Process Analysis Techniques

## Key Improvements
- Reduced resolution time
- Improved SLA compliance
- Automated ticket routing
- Better customer experience

- KPI Analysis
